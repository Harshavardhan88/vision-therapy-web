### Option 1: One-Click Start (Windows)
1.  Double-click **`start_dev.bat`** in the root folder.
2.  This will open two terminal windows (Backend and Frontend).
3.  Open your browser to: [http://localhost:3000/therapy](http://localhost:3000/therapy)

### Option 2: Manual Start
**Backend:**
```bash
cd backend
pip install -r requirements.txt  # If not installed
uvicorn main:app --reload
```

**Frontend:**
```bash
# In the root folder
npm install  # If not installed
npm run dev
```
